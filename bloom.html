<  Bloom  />

Project

Design Brief

Create a static or dynamic data visualization as a set of glyphs. Define a limited number of tasks that the user should be able to complete from the the visualization.

  • Identify range of bloom dates across centuries
  • Identify if peak bloom differs between centuries
  • Compare changes in data collection methods over time
  • Identify bloom dates which fall outside of the norm
